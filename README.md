# Happily - Landing Page

Una landing page limpia y moderna para asesoría pedagógica y psicológica.

## Tecnologías

- **React 19** - Framework UI
- **Vite** - Build tool
- **Tailwind CSS** - Estilos
- **Lucide React** - Iconos

## Estructura

```
src/
├── components/
│   ├── Navbar.jsx      # Navegación responsive
│   ├── Hero.jsx        # Sección principal
│   ├── Services.jsx    # Servicios ofrecidos
│   ├── Psychology.jsx  # Detalle de psicología
│   ├── Contact.jsx     # Formulario de contacto
│   └── Footer.jsx      # Pie de página
├── App.jsx             # Componente principal
├── index.jsx           # Punto de entrada
└── index.css           # Estilos globales
```

## Instalación

```bash
npm install
```

## Desarrollo

```bash
npm run dev
```

## Build

```bash
npm run build
```

## Características

✅ Responsive design (mobile-first)
✅ Formulario de contacto funcional
✅ Navegación suave (smooth scroll)
✅ Menú móvil responsive
✅ Animaciones CSS
✅ Sin librerías complejas (UI components)
✅ Código limpio y bien estructurado
